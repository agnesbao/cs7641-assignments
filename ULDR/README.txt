Please find the code at https://github.com/agnesbao/cs7641_assignments/tree/master/ULDR

This project uses the Fashion-MNIST dataset. To access the data in the project:
```
cd assignment3
git clone https://github.com/zalandoresearch/fashion-mnist.git
```

This project is writen in python 3. Python dependencies are in `assignment3/requirements.txt`
The other dataset used in the project are in `assignment3/data/`
To reproduce the figures in the report, simply run
```
python assignment3/run_all.py
```
It will save figures to `assignment3/output/`
